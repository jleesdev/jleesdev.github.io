On The Line: Puzzle Game — How can we help you?

How to Play

The goal of the game is simple: connect objects and clear them to score points.

Basic Rules

  • Connect 5 or more identical objects in a horizontal, vertical, or diagonal line to clear them.
  • Each time you move an object, 3 new random objects appear on the board.
  • If objects are cleared during a move, no new objects will appear for that move.
  • Objects cannot move through paths blocked by other objects.
  • If there is no space left to place new random objects, the game ends.

Undo Move

If you make a mistake, you can undo your last move.

Tap the back arrow at the top-left corner and watch a rewarded ad to undo your previous action.

Scoring System

When objects are cleared, the score depends on how many objects are connected.

If n objects are cleared:

Score = n(n − 4)

Sometimes the 3 randomly spawned objects may accidentally form a line and clear automatically.

When this happens:

  • The same scoring rules apply
  • The combo multiplier also applies

Game Over

The game ends when:

  • You move an object
  • New random objects must appear
  • But there is no empty space left on the board

When that happens, the game is over.
